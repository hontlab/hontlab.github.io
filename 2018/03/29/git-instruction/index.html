git 명령어 정리

2018-03-29
#명령어 #git
git

git 최초 사용을 위한 세팅

git init

원격저장소 정보 세팅

git remote add origin (원격저장소 url)

원격저장소 정보 수정

git remote set-url origin (수정할 원격저장소 url)

원격저장소에서 다운로드 할 경우

최초

git clone (원격저장소 url)

이후

git pull

git 설정 정보 확인

git config --list

git 업로드 전 수정 및 삭제 파일 감지

git status

로컬에서 수정파일 add

git add (수정파일명 or *[모든 파일])

로컬에서 수정파일 commit

git commit -m "메시지 내용"

커밋 전에 add하여야 커밋 가능

git 업로드

git push -u origin master

svn의 commit과 달리 push까지 해줘야 실제 원격저장소에 업로드
